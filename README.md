# 1.簡介
使用OpenCV尋找畫面中的臉部

# 2.安裝指南
### 系統要求
- Unity 2022.3.33f1 或更高版本。
### 安裝指南
打開Unity並打開這個項目。
需要配備攝影機。
  
# 3.使用說明
### Assets/Scenes/SampleScene.unity
開啟攝影機，讓臉部出現在畫面中。
 
# 4.目標和功能
### 目標
- 演示OpenCV在臉部偵測上的應用
### 功能
- 使用OpenCV對畫面做臉部偵測
